A plugin that records statistics for a WordPress site.

== Description ==

A lightweight plugin that records and displays statistics about visitors to the WordPress site on which the plugin is installed.

= Credits =

Inspiration comes from ( christosoft.de ) CrazyStat php script. Drawing from the idea of a script that records basic information about site visit statistics and then displays them in a single admin page.

Install like a normal wordpress plugin. Like by pasting the plugin directory into the WordPress plugins directory.
